# 6. package

사용 패키지(모듈)은 다음과 같습니다.

- **crypto** : 패스워드 암호화 및 인증 
- **express** : 웹,서버 개발 프레임워크
- **jsonwebtoken** : JWT(Json Web Token) 생성 및 인증 
- **multer** : 파일 업로드 도구
- **multer-s3** : AWS S3 파일 업로드 도구
- **rand-token** : 랜덤 토큰 생성 도구
